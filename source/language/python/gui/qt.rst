Python Qt编程
=========================

Python使用 Qt 有两个库：

* PyQt
* Pyside

两种库都是基于 c++ 的 Qt，所以类、方法都差不多

PySide
###########################

Qt for Python为Qt提供官方Python绑定，并有两个主要组件：

* PY Side6，使您可以使用QT6 AP在您的Python应用程序中，并且

* Shiboken6，绑定生成器工具，可用于将C ++项目公开到Python，以及具有一些实用程序功能的Python模块。

该项目可在LGP LV3 / GP LV3和QT商业许可证下提供。
